# Healthcare App

This repository is organized into two top-level directories:

```
healthcare/
├── frontend/   # React + Vite + TypeScript application
└── backend/    # Backend (placeholder – not yet implemented)
```

## Frontend

The frontend is a React application built with Vite, TypeScript, Tailwind CSS, and shadcn/ui.

### Getting started

```bash
cd frontend
npm install
npm run dev       # start development server (http://localhost:8080)
npm run build     # production build
npm run preview   # preview production build locally
npm run test      # run unit tests (Vitest)
npm run lint      # lint the codebase
```

## Backend

The backend is not yet implemented. See [`backend/README.md`](./backend/README.md) for details.
